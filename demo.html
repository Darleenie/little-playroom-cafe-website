How It Works

This system automatically loads content from your content/website-content.txt file and populates your HTML pages dynamically. Here's how:

Step 1: Edit Content File

Make changes to content/website-content.txt

# Example: Change the hero title HERO SECTION: - Main title: "LITTLE PLAYROOM CAF√â - NOW OPEN!"

Step 2: HTML Uses Data Attributes

Your HTML elements have data-content attributes that the system recognizes:

<h1 data-content="main_title">LITTLE PLAYROOM CAF√â</h1>

Step 3: JavaScript Automatically Updates

The content-loader.js script reads the content file and updates all matching elements.

Benefits

  • Single Source of Truth: All content in one file
  • Easy Updates: No need to edit HTML files
  • Consistent Content: Changes apply across all pages
  • Static Website Friendly: Works with Netlify, GitHub Pages, etc.
  • No Database Required: Just a text file
